# Git Playground

Small repo to practice **Git** with tiny Markdown notes + simple Python scripts.

## What I practice here
- Git basics: status, add, commit, log
- Branching: create/switch, merge
- Conflicts: create and resolve
- Rebase: clean history (interactive rebase practice)
- GitHub flow: push, Pull Request, merge, delete branches

## Project structure
- `docs/` — short notes (Markdown)
  - `01-basics.md`
  - `02-branches.md`
  - `03-rebase.md`
- `sandbox/` — small Python scripts
  - `hello.py`
  - `calc.py`
